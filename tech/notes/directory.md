# Some of my rules on directory organization in windows pc

## Large Disk (HDD)

- Cache
  - GoogleDrive
  - OneDrive
- Documents
  - Markdowned
  - Soft -> ~/Documents
  - Backup
  - Other
- Downloads -> ~/Downloads
- Family
- Games
  - Files
  - Mods
  - Saves
- Videos -> ~/Videos
  - Captures
  - Series
  - Movies
- Music -> ~/Music
- Pictures -> ~/Pictures
  - Backgrounds
  - Generated
  - Screenshots
- Temp

## Fast Disk (SSD)

- Applications
- Dev
- Games
